Table Events ​

名称说明类型
paginationChange分页改变触发[Function] (pageInfo: PageInfo) => void

clickActionConfirmCancel点击操作栏需要二次确认的取消时触发ButtonsCallBackParams
dragSortEnd拖拽排序列图标触发[Function] ({newIndex: number, oldIndex: number}) => void

.........
