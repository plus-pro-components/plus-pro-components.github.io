@plus-pro-components/utils ​

简介 ​

@plus-pro-components/utils 会随着主包一起下载，无需单独安装。这是一个独立的 npm 包，也可以独立安后用于其他项目。

使用示例

ts
import { formatMoney } from '@plus-pro-components/utils'

formatMoney('100')
