@plus-pro-components/eslint-config ​

简介 ​

@plus-pro-components/eslint-config 是 plus-pro-components 内置的 eslint 配置集合，集成了常用的 vue、ts、import插件，为快速配置 eslint 提供保障。

使用 ​

  • 安装

    sh
    pnpm i @plus-pro-components/eslint-config
    1

  • 配置

    js
    // .eslintrc.js
module.exports = {
  root: true,
  extends: ['@plus-pro-components/eslint-config']
}
